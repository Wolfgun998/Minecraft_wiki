Minecraft java

Minecraft: Java Edition (formerly just Minecraft) is the original version of Minecraft, developed by Mojang Studios for Windows, macOS, and Linux.
Notch began development on Minecraft on May 10, 2009, and publicly released it a week later on May 17, 2009.
The full release (version 1.0.0) was released on November 18, 2011, at MINECON 2011.

Some features:

  • Brewing
  • Mobs
  • Ores
